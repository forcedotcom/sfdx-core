Class OAuth2WithVerifier

Extend OAuth2 to add code verifier support for the auth code (web auth) flow const oauth2 = new OAuth2WithVerifier({ loginUrl, clientSecret, clientId, redirectUri });

const authUrl = oauth2.getAuthorizationUrl({ state: 'foo', prompt: 'login', scope: 'api web' }); console.log(authUrl); const authCode = await retrieveCode(); const authInfo = await AuthInfo.create({ oauth2Options: { clientId, clientSecret, loginUrl, authCode }, oauth2}); console.log(access token: ${authInfo.getFields().accessToken});
