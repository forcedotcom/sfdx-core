Class ScratchOrgCache

A Time To Live configuration file where each entry is timestamped and removed once the TTL has expired.

Example

import { Duration } from '@salesforce/kit';
const config = await TTLConfig.create({
  isGlobal: false,
  ttl: Duration.days(1)
});
